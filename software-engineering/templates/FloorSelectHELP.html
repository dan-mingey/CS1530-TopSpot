    Floor Select Help

    The top of the Floor Selection page contains a navigation bar at the top. The first element is a "Help" button (which lead you to this page). The button to the right of the Help button (circled above in red) leads you to a page which allows you to add and manage your friends. Clicking on this button will take you there.

    The right-most element at the top of the navigation bar is a logout button. Clicking on this button will log you out of the TopSpot. You will be prompted to login again if you wish to enter the application again.

    Below the TopSpot logo there are 5 buttons you may choose to select. Each of these buttons represents a floor in the library. Clicking on each of these buttons will take you to a map of the desired floor.

    Each of the floor plan buttons contains capacity information about each of the floors. The first number shown represents how many of the seats are occupied on that floor and the second number represents how many seats total there are on the floor.

    We hope this was helpful. Please hit the "Back" button in the top bar of the screen to go back to the Floor Select page!
